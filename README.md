## 🌸 Sobre este repositório

Este repositório reúne a resolução de **3 atividades práticas** envolvendo criação de banco de dados, tabelas, inserção de registros e consultas filtradas com `SELECT`, `WHERE` e `UPDATE` em MySQL.

## 🌸 Habilidades trabalhadas

- Criação de banco de dados
- Criação de tabelas
- Inserção de registros
- Consulta filtrada de dados (`SELECT` + `WHERE`)
- Atualização de registros (`UPDATE`)

---

## 🌸 Estrutura do repositório

```
🌸 exe01.sql   →  Atividade 01 – Banco de RH
🌸 exe02.sql   →  Atividade 02 – Banco de E-commerce
🌸 exe03.sql   →  Atividade 03 – Banco de Registro Escolar
🌸 README.md
```

## 🌸 Atividade 01 — Banco de RH

> Banco `db_rh` com a tabela `tb_colaboradores`, contendo nome, cargo, salário, data de admissão e departamento.

- Insere no mínimo 5 colaboradores
- `SELECT` de colaboradores com salário **maior** que 2000
- `SELECT` de colaboradores com salário **menor** que 2000
- `UPDATE` de um registro específico
